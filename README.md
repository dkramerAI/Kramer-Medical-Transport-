# Kramer Medical Transport Website

Professional BLS ambulance transportation website serving Stanislaus County and all of California.

## Features

- **4-Page Website**: Home, Services, About, Contact
- **Responsive Design**: Mobile-friendly layout
- **Quote Request Form**: Online transport booking
- **FAQ Section**: Interactive accordion
- **Testimonials**: Client reviews
- **Service Area Map**: Embedded Google Maps
- **EMT Credentials**: Team qualifications display

## Pages

- `index.html` - Home page with hero, features, testimonials
- `services.html` - Services, FAQ, fleet gallery, payment options
- `about.html` - Company info, mission, EMT credentials
- `contact.html` - Contact info, quote request form

## Running Locally

```bash
npx serve .
```

## Deployment

This site is deployed on [Vercel](https://vercel.com).
